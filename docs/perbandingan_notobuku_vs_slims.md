# Tabel Perbandingan Nilai Notobuku vs SLiMS

| Aspek                | Notobuku                                      | SLiMS                                         |
|----------------------|-----------------------------------------------|-----------------------------------------------|
| Fokus                | Exporter MARC21/RDA, interoperabilitas, AI    | Manajemen perpustakaan open source, komunitas |
| Fitur Utama          | Export MARC21 lengkap, media profiles, AI     | Bibliografi, copy cataloguing, laporan, USB   |
| Compliance           | Validasi metadata, matrix uji, UAT checklist  | Standar operasional, komunitas                |
| Interoperabilitas    | Koha, Alma, WorldCat, ILS lain                | Union Catalog Server, federated search        |
| AI & Otomasi         | Pustakawan digital, rekomendasi, integrasi API| Tidak ada AI, fitur dasar lengkap             |
| Portabilitas         | Docker, Laravel, API eksternal                | Portable (USB), multi-platform                |
| Kemudahan Instalasi  | Composer/Docker, dokumentasi lengkap          | Portable, instalasi mudah, gratis             |
| Komunitas            | Pengembangan modern, dokumentasi per role     | Komunitas besar, dukungan luas                |
| Nilai Utama          | Compliance, ekspor MARC/RDA, AI, modern stack | Kemudahan, portabilitas, komunitas, gratis    |

> Pilihan tergantung kebutuhan: Notobuku untuk compliance dan AI, SLiMS untuk kemudahan dan komunitas.
