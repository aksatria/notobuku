# Panduan Peran User

## super_admin
- Governance lintas cabang.
- Monitoring kualitas data dan operasional.
- Kontrol konfigurasi strategis (mis. kebijakan, monitoring SLO, readiness).

## admin/staff
- Operasional harian katalog, sirkulasi, anggota, laporan, serial.
- Menjalankan import/export dan tindak lanjut antrian operasional.
- Verifikasi hasil uji/UAT rutin.

## member
- Pencarian OPAC.
- Pinjam, reservasi, dan notifikasi.
- Interaksi layanan digital perpustakaan.

## Akses dokumentasi in-app
- Route: `/docs`
- Berisi panduan per modul dan checklist UAT.
