# Fitur Utama

- Export MARC21 lengkap (Leader, 00X, 1XX/7XX, 245, 264, 3XX, 5XX, 6XX, 8XX, 856).
- Profil media berbasis `media_profiles` (audio/video/map/teks/serial).
- Konsistensi audiobook online (006/007 + 33X + 347/538).
- Dedup field 856 berbasis URL.
- Relator term/code otomatis (author/editor/narrator, dll).
- Validasi ekspor (008 panjang 40, kode bahasa 3 huruf, aturan profile).
- Modul ILS inti: katalog, sirkulasi, anggota, laporan.
- Modul tambahan: stock opname, serial issue, copy cataloging, OPAC & interop.
