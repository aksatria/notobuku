# FAQ

## Kenapa halaman `/wiki` di GitHub kosong?
Karena repo wiki belum diisi atau fitur wiki belum pernah dipush kontennya.

## Apakah wiki wajib dipakai?
Tidak wajib. Dokumentasi bisa tetap di `README.md`/`docs`. Wiki dipakai jika ingin navigasi dokumentasi yang lebih nyaman.

## Bedanya docs in-app dan GitHub Wiki?
- Docs in-app (`/docs`) untuk user aplikasi.
- GitHub Wiki untuk dokumentasi proyek/kontributor/operasional teknis.

## Bagaimana update wiki cepat?
Edit file markdown wiki, lalu commit + push ke repo `notobuku.wiki.git`.
