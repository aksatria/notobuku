# Draf GitHub Wiki NOTOBUKU

Folder ini berisi halaman wiki siap upload ke GitHub Wiki:
`https://github.com/aksatria/notobuku/wiki`

## Cara publish ke GitHub Wiki
1. Clone repo wiki:
```bash
git clone https://github.com/aksatria/notobuku.wiki.git
```
2. Salin semua file `*.md` dari folder ini ke folder clone wiki.
3. Commit dan push:
```bash
git add .
git commit -m "Initialize NOTOBUKU wiki pages"
git push
```

## Catatan
- Halaman utama GitHub Wiki wajib bernama `Home.md`.
- Navigasi samping bisa dikontrol lewat `_Sidebar.md`.
- Judul halaman bisa diedit langsung dari heading pertama tiap file.
