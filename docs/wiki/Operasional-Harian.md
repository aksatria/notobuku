# Operasional Harian

## Alur ringkas
1. Katalog: input bibliografi lalu tambah eksemplar.
2. Sirkulasi: pinjam/kembali/perpanjang dan review denda.
3. Anggota: import CSV dengan preview, confirm, lalu undo jika perlu.
4. Laporan: filter periode/cabang lalu export CSV/XLSX.
5. Serial issue: kelola expected/missing/claimed/received.
6. Stock opname: buat sesi, scan barcode, selesai, lalu export audit.
7. Copy cataloging: pilih sumber, cari, impor, lalu review metadata.

## Tips operasional
- Selalu validasi metadata inti (judul, pengarang, subjek, ISBN, DDC).
- Pantau antrian zero-result OPAC secara berkala.
- Simpan bukti readiness dan UAT untuk audit internal.
