# Operasional Harian

## Alur ringkas
1. Katalog: input bibliografi lalu tambah eksemplar.
2. Visitor Counter: check-in/check-out onsite, undo 5 menit, lalu review audit.
3. Sirkulasi: pinjam/kembali/perpanjang dan review denda.
4. Anggota: import CSV dengan preview, confirm, lalu undo jika perlu.
5. Laporan: filter periode/cabang lalu export CSV/XLSX.
6. Serial issue: kelola expected/missing/claimed/received.
7. Stock opname: buat sesi, scan barcode, selesai, lalu export audit.
8. Copy cataloging: pilih sumber, cari, impor, lalu review metadata.

## Referensi cepat
- SOP staf baru: [SOP Harian Pustakawan Pemula](SOP-Harian-Pustakawan-Pemula)
- Panduan detail visitor counter: [Panduan Visitor Counter](Panduan-Visitor-Counter)

## Tips operasional
- Selalu validasi metadata inti (judul, pengarang, subjek, ISBN, DDC).
- Pastikan tidak ada visitor aktif tersisa saat tutup layanan.
- Gunakan export audit CSV/JSON visitor counter untuk kebutuhan rekap dan audit teknis.
- Pantau antrian zero-result OPAC secara berkala.
- Simpan bukti readiness dan UAT untuk audit internal.
